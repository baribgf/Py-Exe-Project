# PyExe v1.0

This is a `py-to-exe` converting tool.

## Usage:

* Interactive mode:
```bash
$ ./py-exe.sh
```

* Direct mode:
```bash
$ ./py-exe.sh <main-python-source> <out-exe-path> <do-not-compile-all-project>
```
### NOTICE:
Host system must have python vX installed to run the distributed package.
